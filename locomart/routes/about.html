    Introduction

    In today’s world cloud computing is key to the success of business. Every organization wants to migrate their business to the cloud. There are many benefits to moving to the cloud.

    Features

    We are providing the virtual private cloud service for a business. This will provide private cloud environment where business can have secure computing. In this VPC, we have setup the active directory which stores all the user credentials in the enterprise. Most of the businesses have multiple application which employees use. For security and ease of use, they need single-sign-on. So multiple applications like Jenkins, Github, custom applications all can be accessible using the same credentials.Commonly all the enterprises use the on-premise directories to store the user credentials. So to provide similar concept we have simple directory setup which stores all the user credentials. All the applications in the system are hosted on the separate EC2 instances. For SSO implementation, any third-party SAML provider can be used like OneLogin, Auth0, Octa etc. This third-party SAML provider is integrated with each application for the authentication. We have provided the Jenkins continuous integration environment for fast deployment of the application. Jenkins have the Github plugin which will build the test cases at defined time interval. We have deployed a shell script which will take the latest pull of the application from GitHub after each successful build and deploy it on the EC2 instance. “Locomart” application is part of the system, which is developed using MEAN stack. SSO implementation is provided for all the applications in the system.
